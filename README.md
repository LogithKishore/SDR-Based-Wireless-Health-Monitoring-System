# SDR-Based Wireless Health Monitoring System

## Overview
This project is an SDR-based wireless health monitoring system for real-time patient monitoring and emergency data prioritization.

## Features
- Heart Rate Monitoring
- SpO2 Monitoring
- Temperature Monitoring
- ESP32-Based Processing
- ThingsBoard Cloud Integration
- Emergency Alert Prioritization

## Hardware
- ESP32
- MAX30100 Pulse Oximeter
- DHT11 Temperature Sensor

## Software
- Arduino IDE
- ThingsBoard Cloud
- MQTT Protocol
